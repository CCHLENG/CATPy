===========
从这里开始
===========

Python 是一门广泛使用的编程语言，语法简单，入门快速；动态类型，易于学习。

.. role:: strike
    :class: strike

我们学习 Python 是一种手段而不是目的，目的是解决我们科研中的问题。\ :strike:`虽然学习 Python 成为一个 996 的码农也不失为一个好的选择`\ 。 

本文档会带你解决一些实际问题，让你学习解决问题的思路，最终写出自己的代码。

基本学习路线
============

- 基础
    - 了解数据类型和基本语法：
        知道常用的 List，Tuple 以及 Dictionary
    - 掌握分支、循环结构的语法：
        - for / while
        - if / else
        - 多层 if / else 的写法
    - 学会调用其他 .py 文件中的函数：
        - import 语句的使用
        - 会调用自己写的 .py 文件中的函数

- 内置函数进行文件读写
    - txt 的读写，了解不同打开模式
        - 如: open('demo.txt', 'a')
        - 其中的 'a' 为追加模式
    - 使用 with 语句来自动关闭文件
    - Pandas 模块可以帮助你读写一些包含汉字/英文的较复杂的csv / txt

.. note::

    非必要情况不建议使用 excel 的 xsl 格式读取/保存数据。
    如需要在 Excel 中画图等，请在 Python 中保存为 csv 格式之后另存为 xsl。

- 科学计算
    - Numpy 及 Scipy 模块
    - 中间结果使用 \*.npy 格式保存，可以加速读写
    - Numpy 有内置的函数 savetxt 和 loadtxt 来读写 txt 文件
    - 具体的函数现用现查即可

- 绘图
    - 使用 Matplotlib
    - 学会参考官网的画廊
    - 常用的操作可以自己封装成函数，直接调用，减小代码量，提高可读性

- 图像处理
    - 使用 OpenCV - Python
    - 注意版本，实验室现有的一些脚本使用的是 3.x.x 版本的包，最新的为 4.x.x，其中有一些函数不一致，需要手动修改。

- 网络通信
    - 内置的 Socket 包收发信息，Struct 解码。
    - 了解 TCP，UDP 大概的原理，学会用 Wireshark 抓包调试。

- 嵌入式硬件开发
    - Pyserial 控制串口
