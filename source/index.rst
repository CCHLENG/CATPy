欢迎来到 CCAT Python 学习文档!
=======================================

这是一个 Python 入门教程，目标是让你快速的掌握 Python 基础并写出能解决你实际问题的脚本。

在科研中我们会遇到一些需要批量处理的内容，批量处理数据、绘图等。这就需要掌握一门编程语言来
解决这些重复性的问题。Python 作为一门应用广泛的“胶水语言”，用户基数大，带来了丰富的社区支持，
可能遇到的困难基本都被前人解决过了，你只需要去搜一搜就可以。

当你掌握了 Python 的基础用法之后，就可以结合一些控制设备把试验自动化，解放你自己的双手，打开新世界的大门。
使用 Python 解决科研中的问题，不用过分追求性能，在可接受的时间内解决了问题就达到目的了。
